## Sobre

Um projeto front-end onde desenvolvi uma tela de login e cadastro animada com jquery e CSS. Um projeto muito gratificante, com uma animação refinada e brilhante. O projeto foi inspirado em um vídeo do canal True Coder do Youtube.

## Tecnologias utilizadas

As seguintes tecnologias foram utilizadas no projeto:

- HTML e CSS
- JavaScript
- JQuery
